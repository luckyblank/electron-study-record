# v1.0.3 更新日志

## 🔔 自定义系统级通知（重磅更新）

学习记录的通知焕然一新——开始、结束、暂停、恢复等事件现在通过**完全自绘的系统级浮层**通知，复刻 app 蓝紫玻璃拟态主题，三套主题（亮色 / 深色 / 护眼绿）自动适配。

- **左侧学习宠物形象**：通知左侧会显示你当前的学习宠物，跟随它的成长阶段变化
- **5 秒倒计时 + 进度条**：通知底部进度条直观展示剩余时间
- **鼠标悬停暂停**：把鼠标移到通知上，倒计时和进度条会暂停；移开后继续——读通知时不会被催
- **右上角关闭按钮**：不想等 5 秒？点右上角 × 立即关闭
- **标题层级清晰**：顶部「学习时间记录」品牌标识 + 事件标题 + 次要信息（如「今日累计 01:23:45」）
- **窗口隐藏到托盘时**自动回退原生系统通知，离开窗口也能看到提醒

## 🎬 GSAP 动画系统重设计

引入 GSAP 动画引擎重写关键动画时刻，告别原先脆弱的计时与重绘 hack，动画更流畅、可中断、时序更可靠。

- **弹窗退出动画**：此前关闭弹窗是瞬间消失，现在补齐了平滑的退出过渡
- **Toast / 宠物庆祝·升级 / 徽章解锁 / 进度环** 全部改为 GSAP 驱动
- **进度环数字滚动**：今日完成度百分比会平滑计数动画，不再是跳变
- 尊重系统「减弱动画」偏好设置

## 🔔 通知逻辑优化

- **结束记录通知**：现在**仅在**你主动点击「结束记录」或按 `Ctrl+Alt+E` 时才弹通知；`Ctrl+Alt+Q` 安全退出、更新安装时静默结束，不再打扰
- **暂停 / 继续记录**也改为系统级通知，与开始 / 结束保持一致的体验

## 🐛 问题修复

- **任务栏缩略图按钮丢失**：窗口从托盘恢复后，任务栏上的开始 / 暂停 / 结束按钮会消失——已修复，恢复显示时自动重建
- **通知背景透字**：通知卡片背景过透明会透出背后内容——已改为不透明纯色
- **通知宽度**：通知过宽——已收窄到更合适的尺寸
