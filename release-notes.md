# v1.0.1 更新日志

## ✨ 新功能
- **检查更新按钮**：标题栏新增圆底向上箭头按钮，发现新版本时按钮变主题色 + 柔光晕呼吸提示，点击查看 Markdown 渲染的更新说明
- **会话备注**：学习历史卡片底部可点击添加备注，支持 500 字，`Ctrl+Enter` 保存 / `Esc` 取消
- **会话标签绑定**：历史卡片点击标签区域弹出多选浮层，逐个点击即增量保存，无标签时显示虚线占位
- **数据统计**:累计时长 / 总记录数 / 最长连续从数据库实时计算
- **徽章解锁系统**:首次学习 / 连续 7 天 / 累计 100h 等 8 种成就，自带解锁动画
- **数据导出**:支持 CSV / JSON 两种格式
- **主题切换**:浅色 / 深色 / 护眼绿三套主题

## 🎨 体验优化
- **整体字号 / 间距重构**:建立 7 档字号系统，主次层级更清晰
- **Toast 重新设计**:弱化为胶囊形 + 彩色小圆点，不再喧宾夺主
- **标签区整行可点**:hover 时整行高亮提示，不只是按钮可点
- **导出页面**:CSV / JSON 改为并列卡片布局，一眼区分
- **标题栏**:左侧显示「学习时间记录 vX.X.X」,迷你模式自动隐藏
- **迷你窗口按钮放大**:22×22 更易点击

## 🐛 问题修复
- 修复多个子页面「底部按钮被遮挡」的显示问题
- 修复在设置面板打开状态下切到迷你模式空白且按钮不可点的 bug
- 修复时间字段被当作 UTC 解析导致的 8 小时偏移问题
- 修复删除会话后累计统计数据不准的问题

## 🔔 通知频率治理
- 「学习满 1 小时」激励 toast 按日去重,重启不再重弹
- 跨过 2 小时后让位于每 2 小时休息提醒,避免同一时段两条提醒
- 自动更新失败时显示「暂无可用更新源」而不是技术错误

## 🔧 内部改进
- 数据库迁移系统:自动扫描加载 + 启动前自动备份
- 寄语持久化到数据库,支持增删改
- 开发 / 生产数据库分离,本地调试不污染发版数据
- 一键打包 + 上传发版工作流
