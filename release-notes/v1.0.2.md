# v1.0.2 更新日志

## 🐾 学习宠物系统（重磅更新）

宠物会随着学习成长，让你的学习过程更有陪伴感和正向激励。

- **三种宠物**：学霸猫（默认）、努力汪（累计 10h / 连续 7 天解锁）、智慧鸮（5 徽章 / 累计 50h 解锁）
- **状态联动**：开始学习（前倾+睁眼）、暂停（歪头疑惑）、结束（跳跃撒星星+经验飘字）、超 2h 疲惫提醒
- **成长体系**：学习 1 分钟 = 1 EXP，`升级 EXP = 100 + (等级-1) × 50`；等级、心情、能量、亲密度四项属性持久化
- **点击互动**：点击宠物可互动，心形飘出 + 亲密度 +1（每日上限 20 次）
- **能量机制**：学习时消耗能量，暂停时恢复，低能量时宠物疲惫提示
- **宠物弹窗**：设置面板「我的宠物」入口，查看已解锁宠物和切换

## 🆕 新增功能

- **暂停/继续记录**：计时过程中可随时暂停（快捷键 `Ctrl+Alt+P`），暂停期间不计入学习时长；恢复后无缝继续累计
- **时间区间预设**：统计时间区间弹窗新增自然日 / 学习日 / 工作日 / 自定义预设，一键切换常用统计口径
- **自定义确认弹窗**：所有删除操作（记录/标签/寄语）改用应用内确认弹窗，替换浏览器原生 `confirm()` 对话框，视觉风格与系统统一
- **图表悬浮提示**：最近 7 天柱状图和 35 天热力图支持悬浮查看日期与精确学习时长

## 🎨 体验优化

- **设置面板重新分组**：按功能重新组织入口，新增「我的宠物」卡片，分组更清晰
- **设置面板固定头部**：标题和关闭按钮固定在顶部，仅中间内容区域滚动
- **滚动位置重置**：每次重新打开设置面板或弹窗时，滚动条都会回到顶部
- **底部安全留白**：滚动到底部时不再遮挡最后一组设置内容
- **暂停状态全面强化**：计时数字弱化显示；暂停按钮切换为"继续"图标与渐变色反馈；标签栏出现"已暂停"标记；迷你窗口边框呼吸光效；托盘菜单标注暂停状态
- **按钮显隐优化**：开始/暂停/结束按钮根据会话状态智能切换，无操作时自动隐藏无关按钮
- **更新下载进度展示**：下载更新时显示进度条和百分比，不再盲目等待

## 🔧 基础设施改进

- **暂停数据持久化**：`study_sessions` 表新增 `paused_at` 和 `paused_duration` 字段，通过数据库迁移 v002 自动升级
- **宠物状态持久化**：宠物等级/经验/属性通过 `user_config.pet_state` JSON 存储，重启不丢失
- **electron-log 日志系统**：主进程接入 `electron-log`，未捕获异常和 IPC 错误自动落盘，打包后也能排查
- **生产环境 DB 路径修复**：安装版数据库从只读 `Program Files/asar.unpacked` 迁移到可写 `%APPDATA%/study-time-record/`，首次启动自动拷贝种子库
- **全局快捷键扩展**：新增 `Ctrl+Alt+P`（暂停/继续），`Ctrl+Alt+M` / `Ctrl+Alt+T` 补齐主进程注册
- 补充警告色、柔和状态色、遮罩色、动画时长和间距变量，宠物样式完全复用主题变量
- 适配浅色 / 深色 / 护眼绿三套主题
